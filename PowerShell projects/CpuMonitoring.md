# Cpu Usage Monitoring 

This was the first project i built after Learning Powershell

It basically tell you how much of your CPU memory is used i.e logs out the percentage of CPU that is used every 5 seconds to a `log.txt` file

If you don't understand because you're just starting to learning powershell, i'd advice you cover these topics first before attempting the project:

## Learn the Core Concepts

- What is PowerShell?
- Cmdlets (Command-Lets)
- Objects (VERY important)
- The pipeline (|)
- Help system

## Control data and automate small tasks

- Variables (`$var`)  
- Arrays  
- Hash tables  
- Loops (`foreach`, `for`, `while`)  
- Conditionals (`if`, `switch`)  
- Filtering (`Where-Object`)  
- Selecting data (`Select-Object`)  
- Sorting (`Sort-Object`)
